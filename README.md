# Testing files
test_variants.py (synthetic data), test_variants_file_odo_file0.py (data using odometry, accelerometer and pressure sensors), test_variants_file_imu.py (dataset from autonomous vehicle), test_variants_file_imu_sbg.py (dataset from an sbg device)

These files allow one to test 3 variants of filters (integration of gyroscope, MEKF, Rev-MEKF) on respectively various dataset.
